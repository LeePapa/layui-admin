## 特性

1. 基于tp5.1
2. 基于layui前端框架
3. 内置用户和权限管理
4. Form类快速构建表单
5. 包含多种表单组件
6. Table类快速构建列表

## 安装方式

第一步：
```shell
$ composer rquire thans/layui-admin
```
第二步：
```shell
$ php think layuiAdmin:install
```
第三步：
```shell
$ php think migrate:run
```

## 包含组件：

* 富文本编辑框【ckeditor4】
* 文本输入框
* 数字输入框
* TextArea输入框
* 单[多]文件[图片]上传
* 开关
* 下拉单选
* 下拉多选
* checkbox
* 权限树
* ICON选择器
* 日期选择，支持多种格式。支持范围。
